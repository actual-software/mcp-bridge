Gateway URL: https://{{ include "mcp-bridge.fullname" . }}-gateway:{{ .Values.gateway.service.port }}

To access the Gateway service from outside the cluster:
  kubectl port-forward service/{{ include "mcp-bridge.fullname" . }}-gateway {{ .Values.gateway.service.port }}:{{ .Values.gateway.service.port }}
  curl -k https://localhost:{{ .Values.gateway.service.port }}/health

## Health Checks
Gateway health: 
  kubectl exec -it deployment/{{ include "mcp-bridge.fullname" . }}-gateway -- /usr/local/bin/mcp-gateway health

Router health:
  kubectl exec -it deployment/{{ include "mcp-bridge.fullname" . }}-router -- /usr/local/bin/mcp-router health

## Logs
View Gateway logs:
  kubectl logs -f deployment/{{ include "mcp-bridge.fullname" . }}-gateway

View Router logs:
  kubectl logs -f deployment/{{ include "mcp-bridge.fullname" . }}-router

## Scaling
Scale Gateway horizontally:
  kubectl scale deployment {{ include "mcp-bridge.fullname" . }}-gateway --replicas=3

## Configuration
Update configuration:
  kubectl edit configmap {{ include "mcp-bridge.fullname" . }}-gateway-config
  kubectl edit configmap {{ include "mcp-bridge.fullname" . }}-router-config
  kubectl rollout restart deployment/{{ include "mcp-bridge.fullname" . }}-gateway
  kubectl rollout restart deployment/{{ include "mcp-bridge.fullname" . }}-router

For more information, visit: https://github.com/actual-software/mcp-bridge
